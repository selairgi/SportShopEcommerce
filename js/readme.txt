Projet de site e-commerce pour la vente d'outils de sports

Ce projet consiste en la création d'un site web de commerce électronique pour la vente d'outils de sports. 
Le site propose une variété de produits destinés aux hommes, aux femmes et aux enfants, ainsi qu'une catégorie dédiée aux équipements sportifs.
Fichiers inclus : 

----Le projet contient les fichiers suivants :

    home.html : Cette page représente la page d'accueil du site. Les visiteurs peuvent y trouver des informations générales sur le site et découvrir les promotions en cours.

    men.html : Cette page présente les produits disponibles pour les hommes. Les visiteurs peuvent parcourir la sélection d'outils de sports spécifiquement conçus pour eux.

    women.html : Cette page est dédiée aux produits destinés aux femmes. Les visiteurs peuvent y trouver une gamme d'outils de sports adaptés à leurs besoins.

    kids.html : Cette page offre une sélection d'outils de sports pour les enfants. Les visiteurs peuvent découvrir des produits conçus pour les plus jeunes amateurs de sports.

    equipements.html : Cette page présente les équipements sportifs disponibles. Les visiteurs peuvent y trouver des articles tels que des ballons, des raquettes, des gants, etc.

    log.html : Cette page permet aux utilisateurs de se connecter au site en utilisant Firebase. Ils peuvent saisir leurs informations d'identification pour accéder à l'interface utilisateur personnalisée.

    sign.html : Cette page permet aux nouveaux utilisateurs de créer un compte sur le site en fournissant leurs informations personnelles et en choisissant un nom d'utilisateur et un mot de passe.

----Versions du site

Afin de répondre aux besoins de différents types d'utilisateurs, le projet propose deux versions des pages principales.

    Version sans connexion : Ces fichiers correspondent à la première version des pages, où les visiteurs peuvent parcourir et acheter des produits sans avoir besoin de se connecter. Les fichiers home.html, men.html, women.html, kids.html et equipements.html sont inclus dans cette version.

    Version avec connexion : Ces fichiers correspondent à la deuxième version des pages, où les utilisateurs doivent se connecter pour accéder à une interface utilisateur personnalisée. Les fichiers indexuser.html, menuser.html, womenuser.html, kidsuser.html, Equipementsuser.html, log.html et sign.html sont inclus dans cette version.

----Instructions de déploiement

Pour déployer ce projet sur votre serveur web, veuillez suivre les étapes suivantes :

    Assurez-vous que votre serveur web prend en charge l'exécution de fichiers HTML, CSS et JavaScript.

    Copiez les fichiers mentionnés ci-dessus dans le répertoire racine de votre serveur web.

    Si vous souhaitez utiliser la version avec connexion, assurez-vous d'avoir configuré Firebase avec les informations d'identification appropriées.
    Vous devrez mettre à jour les paramètres de connexion dans les fichiers log.html et sign.html avec vos propres identifiants Firebase.

    Accédez au site en utilisant l'URL correspondant à la page d'accueil (home.html). Vous pourrez ensuite naviguer vers les autres pages en utilisant les liens disponibles.
